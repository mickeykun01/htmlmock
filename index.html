CONCEPTS

1) Difference between Div and Span?

    DIV

  1. Div is a Block-Level Element.
  2. A Block Element start in new line.
  3. It occupy full width.

    SPAN

  1. Span is a Inline-Level Element.
  2. An inline Element does not start in new line.
  3. It only take the space accoding to content.

5) Difference between Block and Inline?

    Block

  1. Block-Level Element is always start in new line.
  2. It occupy full width of the screen
  3. Block Element have top & bottom margin.

    Inline

  1. Inline Element doesnt start in new line.
  2. It occupy only space of the text
  3. Inline Element dont have top & bottom margin.

6) Difference between Em and Strong?

    Emphasis (Em)

  • Emphasis Tag is similar to Italic Tag.
  • It changes the feeling of the text when read by the user

    • Strong

  • Strong Tag is similar to bold Tag.
  • It apply some Importance to the content.

7) What are Attributes?

  1. In HTML the attribute are Special Keywords and it changes the behaviour of the page
  2. It adjust the behaviour of html.
  3. The Attributes are case sensitive.

8) Difference between HTML Tags and Elements?

    TAGS

  • Tags have starting and ending point.
  • When there is no content written in the tag, then it known as Tags.

    • ELEMENTS

  • The content present inside the tag is known as Elements.
  • It consist starting , content , ending point
  • elements with no tag is known as empty tag

9) Explain br tags?

  • In HTML br is known as Line Break.
  • br element produce line break in text.
  • It is useful for writing poem and address.

10) What are Semantic tags?

Semantic tags

* A semantic element clearly describes its meaning to both the browser and the developer.

Examples: "form", "tables", "article", "footer", "header", "main", "nav", "section" etc
